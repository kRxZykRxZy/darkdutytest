# Starter Kit War Shooter (Campaign + Team Multiplayer)

This Godot project provides a playable war-shooter sandbox with:

- 3 campaign missions with staged objectives and reinforcements
- Local/LAN/online multiplayer host + join flow
- Team mode (**SHELLSHOCKERS** vs **RUSHTEAM**)
- Expanded loadout (RPG, Assault Rifle, Shotgun, Grenade Launcher, Sniper, LMG, SMG, Battle Rifle)
- Large war-city arena including static vehicle and helicopter battlefield props

## Main Flow

1. Launch game -> **Home UI**
2. Choose **Play Campaign** for mission progression
3. Choose **Local Multiplayer** to host or connect to a match

## Multiplayer Hosting + Online Play

1. Host a server (set port, max players, optional password).
2. For internet play, expose/forward the chosen UDP port on your router/firewall.
3. Share your public IP/domain + port with other players.
4. Other players connect from **Connect to Server**.

## Controls

| Key | Command |
| --- | --- |
| `W A S D` | Movement |
| `Space` | Jump |
| `Left Mouse` | Shoot |
| `E` / Mouse Wheel | Switch weapon |
| `1` `2` `3` `4` `5` | Direct weapon slots |
| `R` | Reload |
| `Right Mouse` | Scope |
| `H` | Heal (medkit) |
| `Esc` | Release mouse |
